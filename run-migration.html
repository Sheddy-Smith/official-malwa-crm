If automatic migration doesn't work, follow these steps:

  1. Open FINAL_MIGRATION.sql file in your project folder
  2. Copy all the SQL content (Ctrl+A, then Ctrl+C)
  3. Go to Supabase SQL Editor
  4. Click "New query" button
  5. Paste the SQL (Ctrl+V)
  6. Click "Run" button or press F5
  7. Wait 1-2 minutes for completion
  8. You'll see "Success. No rows returned" message

After migration: Refresh your application and you'll be able to add Customers, Vendors, Labour, and Suppliers without any errors!
